# hoi4_generated_feature_art — Platform-Agnostic Checklist

**Purpose:** Generate fictional/symbolic art for HOI4 mods — report images, news images, custom feature art, faction emblems.

## Checklist
- [ ] Art style matches the mod's visual identity
- [ ] Dimensions match target usage (report event: 450×200, news event: 800×400)
- [ ] Output format is DDS (BC7/DXT5) or PNG for conversion
- [ ] Image has proper alpha channel if needed
- [ ] No real copyrighted material used without permission
- [ ] Consistent with the mod's era and tone

## Output Format
```
## Generated Art: <art_name>
### Type: report | news | feature | emblem
### Dimensions: W×H
### Format: DDS | PNG
### Status: generated | needs revision
```
