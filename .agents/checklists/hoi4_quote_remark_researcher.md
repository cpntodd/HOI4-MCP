# hoi4_quote_remark_researcher — Platform-Agnostic Checklist

**Purpose:** Find and verify quotes, cultural remarks, slogans, and short references for HOI4 mod text.

## Checklist
- [ ] Quote is historically accurate and properly attributed
- [ ] Source is verifiable (book, speech, document, credible website)
- [ ] Quote is in the public domain or used under fair use for modding
- [ ] Attribution format: "— Name, Year" or "— Source, Date"
- [ ] Quote length is appropriate for UI (event text: 2-4 sentences max)
- [ ] Multiple language versions sourced if mod supports translation
- [ ] No anachronistic quotes (pre-1950 for WWII-era content)

## Output Format
```
## Quote Research: <topic>
### Quote: "..."
### Attribution: Name, Year
### Source: [reference]
### Usage: event title | event desc | decision text | focus desc
```
