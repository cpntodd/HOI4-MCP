# hoi4_asset_source_researcher — Platform-Agnostic Checklist

**Purpose:** Find, verify, and process real/archival images for HOI4 mod assets. Research only — no GFX editing.

## Checklist
- [ ] Source image is public domain, CC-licensed, or properly attributed
- [ ] Image resolution meets minimum requirements (≥156×210 for portraits)
- [ ] Image is historically accurate for the mod's setting
- [ ] Source URL and license are documented
- [ ] Image has been processed to target DDS format specs
- [ ] No watermarked or copyrighted images used without permission

## Output Format
```
## Asset Research: <asset_name>
### Source: [URL]
### License: [type]
### Resolution: W×H
### Status: found | needs processing | rejected
```
