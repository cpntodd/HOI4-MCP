# hoi4_improvement_loop_planner — Platform-Agnostic Checklist

**Purpose:** Write deep expansion addenda for features that compile but feel thin, generic, or underdeveloped.

## Checklist
- [ ] Identify what the feature currently does vs what it SHOULD do
- [ ] List missing dimensions: AI behavior, failure states, player choices, visual feedback
- [ ] Design escalation: how does the mechanic deepen over the campaign?
- [ ] Add acceptance criteria: what must be true for the improvement to be complete?
- [ ] Specify new files/systems needed (events, decisions, GFX, localisation)
- [ ] Estimate scope: is this a patch or a rewrite?

## Output Format
```
## Improvement Plan: <feature_name>
### Current State: [1-2 sentences]
### Missing Dimensions: [list]
### Expansion Spec: [detailed plan]
### Acceptance Criteria: [checklist]
### Scope: patch | medium | rewrite
```
