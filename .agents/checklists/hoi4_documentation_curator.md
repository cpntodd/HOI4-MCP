# hoi4_documentation_curator — Platform-Agnostic Checklist

**Purpose:** Reconcile specs, plans, handoffs, manifests, and README files for consistency.

## Checklist
- [ ] All spec files reference the correct implementation files
- [ ] Implementation matches the latest spec version
- [ ] No orphaned handoff documents (implemented but not marked complete)
- [ ] README accurately describes current mod features
- [ ] Changelog reflects recent changes
- [ ] Deprecated docs are marked as superseded
- [ ] Cross-references between docs are valid (no broken links)

## Output Format
```
## Documentation Audit
### Specs: N consistent, N outdated, N missing
### Handoffs: N complete, N pending
### README: up-to-date | needs update
```
