# Platform-Agnostic Checklists

These checklists contain the same specialized knowledge as the `.codex/agents/*.toml` files, but in a format any AI agent can execute inline — no subagent spawning required.

## Usage
1. Identify which specialized check you need (e.g., "audit this focus tree")
2. Read the relevant checklist below
3. Execute each item yourself — read files, verify conditions, report findings
4. The parent agent integrates results
